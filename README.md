# Julian Hsieh Music

Portfolio website for Julian Hsieh — violinist based in the San Francisco Bay Area.

Built by [Bernard Sheng](https://github.com/bunardsheng).

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- Framer Motion

## Development

```sh
git clone https://github.com/bunardsheng/julian-hsieh-music.git
cd julian-hsieh-music
npm install
npm run dev
```

## Deployment

Deployed via [Vercel](https://vercel.com).
